## Description
This project was part of the JavaScript 1 course at Noroff. 
Using my new skills writing JavaScript, create an interactive online store to display product from an API endpoint. Use one of the three provided API URLs to display products in HTML and allow users to create a basket of items. 

**Key Objectives:**

- Dynamic Data Fetching: Integrating a REST API to fetch product data instead of hardcoding content in HTML.

- DOM Manipulation: Dynamically rendering product lists and individual product detail pages based on the API response.

- State Management (Shopping Cart): Implementing a functional shopping cart where users can add/remove items, with data persisting through local storage or session handling.

- Filtering & Sorting: Creating interactive features to allow users to filter jackets by category or price.

- Asynchronous Programming: Utilizing fetch, async/await, and error handling to ensure a smooth user experience during data loading.

**User Stories**

- As a user, I want to view a list of products on the homepage.
- As a user, I want to filter products by category, gender or genre.
- As a user, I want to view a single product page with more detail.
- As a user, I want to add a product to my basket.
- As a user, I want to remove a product from my basket.
- As a user, I want to view a summary of my cart on the checkout page.
- As a user, I want to view an order-confirmation screen after checking out.

**Required Pages**

- Home Page containing product list `/index.html`
- Product Page showing all details of a specific product `/product/index.html`
- Checkout Page showing all items in the basket `/checkout/index.html`
- Confirmation Page showing a thank you message `/checkout/confirmation/index.html`

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone https://github.com/aw3stm/rainyDays_js.git
```

2. Install the dependencies:

```
npm install
```

### Running
To run the app, run the following commands:

```bash
npm run start
```
